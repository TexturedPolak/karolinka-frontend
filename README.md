# karolinka-frontend
Prosty interfejs do zaszyfrowania lub odszyfrowania tekstu szyfrem karolinka, bazujący na bibliotece karolinka, napisanej w pythonie.

Dostępny tutaj: https://texturedpolak.github.io/karolinka-frontend/

Wykorzystuje [PyScript](https://pyscript.net/) do uruchamiania [biblioteki karolinka](https://pypi.org/project/karolinka/).

Jest to swego rodzaju demo działania tej biblioteki.
